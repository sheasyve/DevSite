AsciiRT - Real-time ray-traced animations in the terminal.

AsciiRT is a highly scalable and optimized GPU ray-tracing engine utilizing CUDA to render images from object models and display them in ASCII characters. The application is has been extensively optimized to achieve a 900x speedup over the initial multi-core version, enabling real-time animations.

The demo shows the rendering of a car with ASCII characters by the program, with a runtime of 0.01665 seconds per frame. This mesh of a Toyota Mark II contains 18.3 thousand triangles and 11.2 thousand vertices in object model format. The object model was rotated and converted to text every frame with the ray-tracing engine. The image was generated with reflections and shadows using perspective projection, and four light sources to simulate a realistic looking image in text format.

Users can load an arbitrary number of object files (.obj), and render a picture or an animation. The program will convert the given object files to meshes, perform translations and rotations, calculate output with the ray tracing kernel, and render to the terminal in ASCII with a brightness to character map. Animations can be rendered at 60 frames per second (with the car model), creating smooth output.
